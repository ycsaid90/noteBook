# How to install Playwright

To install Playwright, you can use npm (Node Package Manager) or yarn. Here are the steps to install Playwright using npm:
1. Open your terminal or command prompt.
2. Navigate to your project directory where you want to install Playwright.
3. Run the following command to install Playwright:
````bash
npm init playwright@latest
````

4. After the installation is complete, you can verify that Playwright is installed by running:
```bash
npx playwright --version
```
This will display the version of Playwright that you have installed. You can now start using Playwright in your project to automate browser interactions and perform end-to-end testing.

5.Update latest version of Playwright Test
````bash
npm init -D playwright/test@latest
````
